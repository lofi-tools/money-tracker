# Crypto Tracker

Track crypto across accounts, chains, apps, platforms.
Estimate historical value and current revenue.


## Developer quickstart

Setup deps using `direnv allow` (needs Nix and nix-direnv), then:
- Run unit tests: `utest`
- Run program: `run`
